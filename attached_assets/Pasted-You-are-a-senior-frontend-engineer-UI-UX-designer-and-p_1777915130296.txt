You are a senior frontend engineer, UI/UX designer, and product designer.

Build a production-quality landing page for a startup web platform called “Universal Productivity Zone”.

The landing page must clearly explain the product before the user logs in. It should feel like a modern SaaS/startup website, not a simple template.

PROJECT IDEA:
Universal Productivity Zone is an all-in-one productivity ecosystem where users can learn skills, find jobs, collaborate with teams, manage tasks and projects, communicate through chats, grow as freelancers, and use AI assistants.

The platform personalizes the user experience after registration based on profession, goal, and experience level.

DESIGN DIRECTION:
Use a Modern Minimal Light design.

Visual style:
- Soft milk-white background: #F7FAFC
- White cards: #FFFFFF
- Dark main text: #111827
- Secondary text: #6B7280
- Light borders: #E5E7EB
- Accent colors: #6366F1 and #3B82F6
- Soft shadows
- Large rounded corners
- Clean spacing
- Apple / Notion / modern SaaS inspired
- Premium, emotional, clean, and trustworthy
- Avoid clutter
- Avoid too much color
- Avoid aggressive animations
